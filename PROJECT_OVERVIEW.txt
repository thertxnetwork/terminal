MainActivity Features:
  ✓ Sets up Material Design 3 UI
  ✓ Detects current theme mode
  ✓ Implements theme toggle switch
  ✓ Switches between light/dark themes dynamically
  ✓ Handles button click events

Theme Switching Logic:
  • User toggles SwitchMaterial
  • App detects isChecked state
  • Calls AppCompatDelegate.setDefaultNightMode()
  • System recreates activity with new theme
  • UI components automatically adjust colors

Prerequisites:
  • Android Studio (latest version)
  • JDK 21

Build Debug APK:
  ./gradlew assembleDebug

Build Release APK:
  ./gradlew assembleRelease

Install on Device:
  ./gradlew installDebug

Clean Build:
  ./gradlew clean build
